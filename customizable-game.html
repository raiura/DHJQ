📖 游戏API文档

快速开始

你可以通过编辑上方的HTML和CSS来完全自定义你的游戏界面。系统提供了一系列JavaScript API供你调用，让你的自定义页面能够与AI对话、加载角色、获取世界书等功能交互。

🚀 基础API

发送消息

// 发送消息给AI await GameAPI.sendMessage('你好，我是...'); // 流式输出回调 await GameAPI.sendMessage('你好', { onStream: (text) => { // 实时更新AI返回的文本 document.getElementById('dialogue').textContent = text; } });

获取角色

// 获取所有角色 const characters = await GameAPI.getCharacters(); // 根据关键词匹配角色 const matched = await GameAPI.matchCharacters('小明');

世界书

// 获取所有世界书条目 const entries = await GameAPI.getWorldbook(); // 根据文本匹配世界书 const matched = await GameAPI.matchWorldbook('落星谷');

🎨 视觉API

设置背景

// 设置背景图片 GameAPI.setBackground('https://example.com/bg.png'); // 设置渐变背景 GameAPI.setBackground({ gradient: 'linear-gradient(to bottom, #1a1a2e, #0d0d1a)' });

显示角色

// 显示角色立绘 GameAPI.showCharacter({ id: 'char_1', image: 'https://example.com/char.png', position: 'center', // 'left', 'center', 'right' animation: 'fadeIn' }); // 隐藏角色 GameAPI.hideCharacter('char_1');

📝 完整示例

// 示例：创建一个简单的聊天界面 <div id="my-game"> <div id="bg"></div> <div id="dialogue"></div> <input id="input" type="text"> <button onclick="send()">发送</button> </div> <script> async function send() { const text = document.getElementById('input').value; // 发送消息 const response = await GameAPI.sendMessage(text, { onStream: (chunk) => { document.getElementById('dialogue').textContent += chunk; } }); // 匹配角色并显示 const chars = await GameAPI.matchCharacters(response); if (chars.length > 0) { GameAPI.showCharacter({ id: chars[0].id, image: chars[0].image, position: 'center' }); } } </script>
